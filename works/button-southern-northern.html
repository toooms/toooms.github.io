Submission for interactive <button> art show

My submission for Northern-Southern's "MAKE A BUTTON FOR ART!" interactive art show. Trying to explore new ways of html buttons appearance. Flower wallpaper from http://www.wallsave.com/

This is a clean HTML5 <button> element, bringing attention to :hover and :active states and showing that good old btns don't have to be office gray and booooring.

Note that this button has been made for an specific art show, screen size and browser, meaning that support on other browsers and different screen sizes can be limited. Few pixel displacement is possible if not using Chrome. I will try to work on fallbacks.
